## What I do
- Design and ship resilient APIs, services, and automation with Java, Python, C#, Node.js, and C.
- Ship full stack products with modern JavaScript frameworks and intentional UI systems.
- Automate builds, testing, observability, and deployments with CI/CD and containers.
- Optimize data flows with relational and NoSQL databases; design caching and streaming where it matters.

## Highlights
- Backend-first mindset: start from the domain, contracts, and data models before pixels.
- Performance curious: profiling, async/concurrency, and backpressure instead of guesswork.

- Pragmatic security and reliability: rate limits, auth flows, and guardrails to keep systems steady.
